== Description ==

Now you can customize the Login/Registration Page with Live Preview using wordpress Customizer feature.

The easiest way to brand your WordPress dashboards the way you want. Easily customize your admin dashboard, change WordPress logo with your company logo in admin panel, login and registration page. 

=== Translation/Localization Ready ===
For translators who wants to add new language to AE Admin Customizer contribute via [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ae-admin-customizer)

=== Features ===
* Customize Admin Dashboard Sidebar, Topbar and Global Colors
* Customize Login and Registration Page with Customizer for Live Preview
* Add Custom CSS Styling on Admin Panel, Login and Registration Pages
* Remove Wordpress Icon on Admin Topbar
* Add Custom Logo to Admin Topbar, Login and Registration Page
* Ability to Remove Wordpress Version No. on Admin Panel
* Ability to Remove Version numbers from scripts and stylesheet's HTML
* Ability to Remove Wordpress Generator Tag both HTML and RSS Feeds
* Ability to Remove Help Tab from Admin Dashboard
* Ability to Remove Screen Options Tab
* Ability to Remove Welcome to Wordpress Dashboard Widget
* Ability to Remove Wordpress News and Events Dashboard Widget
* Change or Replace Admin Left Footer Text with your own 
* Ability to Import / Export AE Admin Customizer Settings

== Installation ==

INSTALLING FROM WITHIN WORDPRESS

1. Visit 'Plugins > Add New'
2. Search for 'AE Admin Customizer'
3. Activate 'AE Admin Customizer' from your Plugins page.

MANUAL INSTALLATION

1. Upload `ae-admin-customizer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= What does this plugin do? =
This plugin lets you customize the admin panel, login and registration pages. Usually useful when you are branding your websites or want your website admin panel to be different from wordpress default dashboard.

= Does this plugin offer a way to customize the login and registration page with live preview? =
Yes, we have included the functionality since version 1.0.1, If you are still using the old version, you'll need to update the plugin to use the new functionality.

= I am having trouble using the plugin and it broke my site for some reason. =
The plugin itself shouldn't break your website upon installation although there may be cases when you're running an old version of wordpress. You'll need to update the wordpress core in order to use this plugin.

= What is the minimum requirements to be able to use AE Admin Customizer? =
* At least wordpress 4.5
* PHP 5.6

= I want to translate your plugin with our language, where should I start? =
You can contribute translating the plugin via [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/ae-admin-customizer)
